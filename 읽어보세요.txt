마이메모의 특징
1. 메모 데이터를 TEXT파일로 저장합니다.
- DB나 고유파일 형식이 아니라 다른 프로그램으로도 볼 수 있고 파일이 깨져서 데이터 전체가 날아갈 염려가 없습니다.
- 기존에 Txt파일에 대해서도 관리가 가능합니다.(데이터 전환 필요 무)
- 분류 체계의 무한한 자유로움 - 디렉토리 기능의 혜택을 입어 단계수 제한이 거의 없고 분류 이동도 자유롭습니다.
2. 작성한 메모는 자동으로 저장됩니다.
- 다른 메모로 전환 할때나 종료할 때 자동 저장됩니다.
- 혹시나 모를 시스템 다운 등에 대비하여 저장버튼을 눌러 수동저장할 수도 있습니다.
3. 다수의 메모디렉토리 지정 가능
- 내 컴퓨터 또는 네트워크 환경의 여러 디렉토리를 메모 디렉토리로 지정할 수 있습니다.
4. 디렉토리에도 메모를 달 수 있습니다.
- 이기능과 다수 메모디렉토리 지정기능을 이용하면 단순하지만 디렉토리 관리, 문서 관리 기능으로도 활용할 수 있습니다.
5. 최근 작성한 메모로 쉽게 찾아서 볼 수 있습니다.
- 방금 전에 작성한 메모를 쉽게 찾아서 볼 수 있도록 최근 메모를 8개 까지 기억하고 프로그램을 재시작해도 지워지지 않고 남아 있습니다.


마이메모 시작하기
1. 실행파일(MyMemo.exe)를 임의의 디렉토리에 복사한 후 실행합니다. 처음 시작이면 왼편에 "메모-기본" 아이템이 보일 겁니다.
2. 메모 데이터 디렉토리는 현재 디렉토리가 기본입니다.
3. 새 메모를 추가하시려면 "기본"을 선택하고 오른쪽 마우스 버튼을 눌러 "새 메모"를 선택하면 되고 새폴더를 추가하시려면 같은 방법으로 "새 폴더"를 선택하면 됩니다.
4. 메모 데이터 디렉토리를 다른 곳으로 변경하려면 "기본" 아이템에 오른쪽 마우스 버튼을 눌러 "등록정보" 메뉴 아이템을 선택합니다. 대화상자에서 "경로"를 다른 위치로 변경하면 됩니다.
5. 메모 데이터 디렉토리를 추가할 수 있습니다. 하드 디스크의 여러 곳에 메모를 관리하고 싶으면 최상위인 "메모"에서 오른쪽 마우스 버튼을 눌러 "새 메모 디렉토리"를 선택하여 "이름"에는 표시 이름을 쓰고 "경로"에 지정할 디렉토리를 입력하면 됩니다.

그럼 나머지 기능은 사용하면서 익혀 보세요.

MyMemo의 미래
.MyMemo는 가능한 가볍고 간편한 것을 원칙으로 할 겁니다. 현재 파일 크기도 80Kbyte 정도 입니다. 기능 위주의 프로그램으로 마이메모는 리소스를 차지 하는 화려함은 추구하지 않습니다.(사실 화려하게 만들 능력이 안되요. ^^;)
.일반 Text파일뿐 아니라 MSWord, HTML, CSV등을 마이메모를 통해 관리하고 편집할 수 있게 할 것입니다. 이 제품이름은 MyMemo Plus가 될 것입니다. 다시 말하면 종합적인 개인 문서관리 시스템으로 태어날 것입니다.
.FTP등 리모트 파일 지원과 동기화 기능- 데이터를 서버에 올려 놓고 자신의 컴퓨터에서 변경하면 자동으로 서버에 변경된 정보가 올라갑니다. 어느 곳에서나 같은 메모를 볼 수 있도록해주는 기능입니다. 또는 백업용도로도 유용하겠죠.
